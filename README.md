# Birds-Eye-Game
2D Bird's eye style resource game


## Features
- Random terrain generation
  - islands
  - land bridges
  - beaches
- Breaking and placing of items
- Auto-sorting inventory
- Dynamic minimap (toggleable)


## Controls
--- | ---
left | 'A'
right | 'D'
up | 'W'
down | 'S'
break | left-click
place | right-click
index inventory | scroll-wheel
toggle inventory | 'E'
toggle minimap | 'M'
